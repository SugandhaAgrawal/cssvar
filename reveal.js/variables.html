CSS Variables Aka CSS Custom Properties

A Variable is an element, feature, or factor that is liable to vary or change.

CSS Variables

var('custom-property-name' [, 'declaration-value'])

CSS Variables

CSS Variable function declaration : var('custom-property-name' [, 'declaration-value']) where 'declaration-value' is a fallback value to be used when the referenced custom property is invalid.

CSS Variables - Fallback

✏ Helpful for A Web Component dev to create an initial design using fallback values, and expose theming “hooks” in the form of custom properties.

  • The chain
    • 									

										color: var(--su-var, red);

										color: var(--su-var, var(--su-new-var, green));

										color: var(--su-var, var(--su-new-var, (var(--su-newer-var), yellow)));

    ✏ This method causes performance issues.It makes browsers render pages more slowly than normal as it takes more time to parse through the variables.
