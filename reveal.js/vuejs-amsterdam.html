    Outstanding Talks

    Unit Testing Vue Components

    Tools:

    • Jest: Java script testing
    • vue-test-utils: the official unit testing utility library for Vue.js
    • jest-serializer-vue: pretty printer for snapshots
    • Compilers/Transformers: vue-loader, vue-jest, babel-jest
    • Code sample: e.g. https://github.com/eddyerburgh/vue-test-utils-jest-example

    Outstanding Talks

    Native mobile app with Vue and NativeScript

    • NativeScript is how you build cross-platform, native iOS and Android apps without web views.
    • Playground for Nativescript
    • NativeScript-Vue is a NativeScript plugin which allows you to use Vue.js to craft your mobile application.
    • Hybrid frameworks allow running same application on Android and iOS devices but use web view for rendering hence cant access native components, whereas Nativescript can.
